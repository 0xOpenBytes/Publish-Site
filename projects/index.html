ios-base

This is a SwiftUI iOS development template with components for app, mock data, network, local storage, navigation, networking, notifications, and tests. It uses OpenBytes libraries for input/output, type transformations, and async tasks.



c

c is a simple composition framework. You have the ability to create transformations that are either unidirectional or bidirectional. There is also a cache that values can be set and resolved.



t

t is a simple testing framework using closures and errors. You have the ability to create a suite that has multiple steps, expectations, and asserts. Expectations can be used to expect one or multiple assertions.



o

o is a simple framework to output to a file, url, the console, or even register notification using UserNotifications. o can also get input from a file, url, or console.
